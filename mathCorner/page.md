# 数学角

***

::: details 解析
示例：(x+2)(x+3)(x+4)=120 求 x

**转化成 `三个连续数相乘` 得到积 120**

```bash
// 第一步 分解
120 = 2×2×2×3×5
// 组合
120=2×2×5×2×3=4×5×6
// 求解
x + 2 = 4
// 最终求解
x = 2
```

:::
